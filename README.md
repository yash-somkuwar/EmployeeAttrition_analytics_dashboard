# 📊 HR Employee Attrition Analysis Dashboard (Excel)

## 🔍 Project Overview

This project focuses on analyzing employee attrition using Excel by performing data cleaning, transformation, and visualization. The goal is to identify key factors influencing employee turnover and present insights through an interactive dashboard.


## 📁 Dataset

* Source: HR Employee Dataset (`employee_attrition.csv`)
* Contains information such as:

  * Age
  * Gender
  * Employee Id
  * Job Role
  * Marital Status
  * Education
  * Attrition Status


## 🧹 Data Processing

* Cleaned missing and inconsistent values
* Converted raw data into structured Excel Table
* Created derived columns:

  * Age Groups (18–25, 26–35, etc.)
* Standardized categorical fields for accurate analysis
  

## 📊 Analysis Performed

Using Pivot Tables and Excel functions:

* Employee distribution by Gender
* Attrition by Department
* Attrition by Job Role
* Attrition by Age Group
* Attrition by Marital Status
* Education-wise Attrition
* Performance Rating analysis


## 📈 Dashboard Features

* Interactive dashboard built using Pivot Charts
* Slicers for:

  * Gender
  * Department
  * Age Group
  * Marital Status
* Key KPIs:

  * Total Employees
  * Attrition Count
  * Attrition Rate
  * Average Age


## 🛠 Tools & Technologies

* Microsoft Excel

  * Pivot Tables
  * Pivot Charts
  * Slicers
  * Conditional Formatting
  * Excel Functions

* Kaggle

  * Dataset sourcing
  * Exploratory Data Analysis (EDA) reference
  * Understanding data patterns and feature relationships
    

## 🎯 Key Insights

* Identified departments with highest attrition
* Analyzed age groups most prone to leaving
* Observed patterns based on marital status and job roles
* Highlighted workforce distribution trends


## 🚀 Conclusion

This project demonstrates how Excel can be used as a powerful data analysis and visualization tool to extract meaningful business insights from raw HR data.
